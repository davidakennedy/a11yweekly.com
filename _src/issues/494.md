## Featured: The WebAIM Million - 2026 update

"For the eighth consecutive year, WebAIM conducted an accessibility evaluation of the home pages for the top 1,000,000 web sites. The evaluation was conducted using the WAVE stand-alone API (with additional tools to collect site technology and category data). The results provide an overview of and insight into the current state of web accessibility for individuals with disabilities as well as trends over time."

Read more of [The WebAIM Million](https://webaim.org/projects/million/).

## Sponsored: Live Wednesday: How to stop ARIA from making your site less accessible

ARIA is supposed to improve accessibility. But the data tells a different story - pages with ARIA average significantly more errors than pages without. The problem isn't ARIA itself. It's adding roles without keyboard support, setting aria-expanded without toggling it, or using role="menu" where a nav element belongs. This Wednesday (April 8, 1pm PT) we're walking through the common mistakes and the decision tree for when ARIA actually helps. Live testing with real sites and NVDA.
